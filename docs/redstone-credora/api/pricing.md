# Pricing

### Consumption

Active Credora Network participants benefit from free access to Credora APIs. For pricing benefits, Reviewers are required to submit 5 attestations per quarter.&#x20;

| Access             | Reviewers | Price        |
| ------------------ | --------- | ------------ |
| Direct Consumption | Free      | $500 / month |

### Redistribution

Redistribution fees are dependent on the metrics being redistributed, and the audience of the relevant partner. Request access to the Credora Platform to discuss details.&#x20;
