# Overview

### Ratings and Risk Metrics Drive Informed Decisions

The Credora Network distributes ratings and risk metrics for direct consumption and for redistribution. Direct consumption is where the consumer is utilizing metrics internally, while redistribution is where the consumer is providing access to a wider audience. Ratings are utilized by for various audiences for different purposes, outlined as follows:

| Business Type  | Consumption | Redistribution |
| -------------- | ----------- | -------------- |
| Asset Managers | ✓           |                |
| Wallets        |             | ✓              |
| Exchanges      | ✓           | ✓              |
| Protocols      | ✓           | ✓              |

Below are example use cases:

* **Asset Managers** consume Risk Ratings for internal risk management, optimization of risk-reward decisions, and demonstrating in-depth analysis for relevant stakeholders.
* **Wallets** help users navigate various DeFi protocols, and improve user experiences by providing third-party risk information.
* **Exchanges** manage internal exposures, including approving and managing specific thresholds for collateral assets.
* **Protocols and Applications** utilize Risk Ratings for optimizing capital allocation managing risk parameters.
