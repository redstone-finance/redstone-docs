# Firm Assessments

In order to receive a Firm Assessment, issuers can Request Access to the [Credora Platform](https://credora.io/). Onboarding requires the completion of an account profile, and submission of relevant entity information. Once complete, issuers can directly engage the Credora Team via the chat application in the below image.

<figure><img src="../../assets/image (2).png" alt="" /><figcaption></figcaption></figure>

Documentation can be securely uploaded via the Credora Data Room feature.

<figure><img src="../../assets/image (3).png" alt="" /><figcaption></figcaption></figure>

The Credora team is available for answering any questions regarding the application of the Token Rating Framework. Through this process, Credora aims to provide guidance on industry best practices, thereby elevating standards throughout the DeFi ecosystem.
